## Getting Started

1.  **Installation:**
    ```bash
    npm install -g q-cli
    ```

2.  **Configuration:**
    Configure your API keys and model preferences interactively:
    ```bash
    q config
    ```
    This command creates a local `.qcli.json` file which stores your settings.

3.  **Invocation:**
    Summon the presence of MABEL:
    ```bash
    q
    ```
    Or, if you prefer, invoke her directly:
    ```bash
    mabel
    ```
