## Getting Started

### 1. The Core CLI
```bash
# Clone the repository
git clone https://github.com/noam-sketch/q.git
cd q

# Install dependencies and build
npm install
npm run build

# Configure your API key
node dist/index.js config

# Enter the Terminal
node dist/index.js chat
```

### 2. Q-Local (Host Agent)
To compile the standalone binaries yourself:
```bash
cd q-local
npm install
npm run build
```
The resulting executables can be run from `q-local/dist/index.js` or compiled using `pkg`.
